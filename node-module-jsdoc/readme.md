# JSDoc 

## Discription

---

JSDoc generates a documentation from your functions and converts it to HTML. This setup uses better-docs, so it's optimized for react.

---


### Setup

---

> git clone git@github.com:dominikhaid/jsdoc.git my-app
> 
> cd my-app
> 
> npm run i
> 
> npm run docs
>
